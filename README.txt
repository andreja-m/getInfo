A simple script for collecting informations about pc specifications, such as: user informations, hardware info, storage, network, sockets, etc. Save those informations in semetric encripted passphrased .txt.gpg file.

plan is to add also sending that file to a mail and deleting file after that.

script collect just non-sudo informations, to download and start script open terminal and type:
- git clone https://github.com/doomguy-369/GetInfo
- cd GetInfo
- chmod +x inf.sh (just first time)
- ./inf.sh
- enter password 2 times
- you have 123.txt.gpg file in GetInfo directory with some informations.
